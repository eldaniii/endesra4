Examen Git y Github

Alumn@: Daniel Vilar Martínez

  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones es una herramienta software que permite a los equipos de desarrollo gestionar los cambios que se van realizando en un proyecto a lo largo de su desarrollo.

  2. Cual es la diferencia entre git y github?

    La principal diferencia es que GIT es un software de control de versiones y GITHUB es una plataforma que hace uso de git y otras herramientas para crear un entorno en la nube donde sea posible crear y mantener software e interactuar con otros desarrolladores.

  3. Cómo creo una rama en git?

    Utilizamos en nuestro repositorio el comando "git branch nombre_rama" para crear una nueva ramificación.

  4. Cómo clono un repositorio?

    Un repositorio se puede clonar utilizando un URL o mediante el uso del protocolo SSH. Para clonar un repositorio mediante una URL utilizamos el comando "git clone url_repositorio".

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    Para traer del repositorio en la nube los últimos cambios se debe usar el comando "git pull [alias] [rama]".

  6. Cómo puedo generar un tag?

    Para generar un tag utilizamos el comando "git tag [nombre-etiqueta]"

  7. Cómo hago una mezcla de ramas?

    Utilizando el comando "git merge [nombre-rama]".

  8. Cómo puedo moverme a un commit especifico?

    Podemos trasladarnos a un commit especifico de una rama mediante el uso del comando "git checkout hashcode_nuestrocommit".

  9. Cómo puedo borrar una rama?

    Para borrar una rama localmente podemos hacer uso del comando "git branch -d [nombre-rama]". Para eliminar completamente la rama tenemos que usar el comando "git push origin" en el caso de que ya hayamos mandado la rama a GitHub.

  10. Cómo puedo borrar un tag?

    Para borrar una etiquieta usamos el comando "git tag -d [nombre-etiqueta]".

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    Haciendo uso del comando "git status"

  12. Para que sirve el comando git remote -v?

    El comando "git remote -v" permite crear, ver y eliminar conexiones a otros repositorios mediante una URL.

  13. Cómo le digo a git cual es mi email y correo?

    Utilizando el comando "git config":
    git config --global user.name "Tu nombre"
    git config --global user.email "tuemail@example.com"

  14. Cuales son los pasos para conectar git con github?

    Debes configurar tu nombre y tu correo, utilizando los comandos config anteriores. Y hacer uso del comando "git init" para inicializar un repositorio y "git clone" para clonar uno existente en tu máquina local.

  15. Cómo puedo crear un alias en git?

    Se puede crear un alias en git con git config --global alias.[nombre-alias] ["serie_comandos"]
    Un ejemplo sería: git -config --global alias.tree "log --graph --decorate --all --oneline"
